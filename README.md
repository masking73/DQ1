# DQ1
初代『ドラゴンクエストI』（FC版）の「ふっかつのじゅもん」は、名前、所持金、経験値、武器・防具、アイテムなどのステータスを複雑な暗号化アルゴリズム（チェックサムやビット演算）で20文字の文字列に変換したものです。

本物の仕様を完全に再現すると膨大なコード（バグのないビット処理）が必要になるため、今回は「いかにもドラクエ1でありそうな、文字規則に沿った『ふっかつのじゅもん』を毎回ランダムで生成するWebアプリ」（HTML/JavaScriptの単一ファイル）を作成しました。

ドラクエ1の「ふっかつのじゅもん」で実際に使われている「54文字（濁点・半濁点含む）」の文字セットを使用し、本物らしい20文字の呪文を出力します。
